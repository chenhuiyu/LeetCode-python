# [20. 有效的括号](https://leetcode-cn.com/problems/valid-parentheses/)

## 1. 题目

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s ，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

> 示例 1：
>
> 输入：`s = "()"`
> 输出：`true`
>
> 示例 2：
>
> 输入：`s = "()[]{}"`
> 输出：`true`
>
> 示例 3：
>
> 输入：`s = "(]"`
> 输出：`false`
>
> 示例 4：
>
> 输入：`s = "([)]"`
> 输出：`false`
>
> 示例 5：
>
> 输入：`s = "{[]}"`
> 输出：`true`

## 2. 思路

遍历括号字符串中的每一个字符：

- 遇到左括号: 压栈
- 遇到右括号
  - 如果栈不空：弹栈
    - 如果弹栈的左括号和当前右括号不匹配，return false
  - 如果栈空：不匹配

如果最后栈空，括号匹配，如果栈不空，不匹配

## 3. Code

```python
class Solution:
    def isValid(self, s: str) -> bool:
        stack=[]
        map_dict={
            ")":"(",
            "]":"[",
            "}":"{"
        }
        for char in s:
            # 遇到左括号，压栈
            if char in map_dict.values():
                stack.append(char)
            # 遇到右括号，弹栈
            else:
                # 如果栈空，不匹配，return false
                if len(stack)!=0:
                    left=stack.pop()
                    # 如果弹栈的左括号和当前右括号不匹配，return false
                    if left!=map_dict[char]:
                        return False
                else:
                    return False
        # 最后栈为空，括号匹配
        if len(stack)==0:
            return True
        else:
            return False
```
